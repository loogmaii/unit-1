# Quiz 5

Given a number, create a program that produces the output factors.

## code

```py
n = int(input("enter an integer: "))
factors = [1]
factor = 2
sum = 1

while factor <= n / 2:
    if n % factor == 0:
        factors.append(factor)
        sum = sum+factor
    factor += 1

print(factors)
print(sum == n)
```
