# Quiz 14

Create a function that receives one input and produces the output shown. 

out = mysteryTwo(A=10, B=4)

print(out)

## code

```py
def mysteryTwo(A:int,B:int)->int:
    out = ((A * A) + B)
    return out

out = mysteryTwo(37,3)
print(f'{out}')

out = mysteryTwo(58,2)
print(f'{out}')

out = mysteryTwo(60,5)
print(f'{out}')

out = mysteryTwo(10,4)
print(f'{out}')                                                                             
```
