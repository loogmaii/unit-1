*Fig.1* Solution to the quiz

In Fig.1 I solved the quiz by creating the steps for the black box

## code

```py
a = input("Input text: ")
a = a.split()
output = ""
for word in a:
    if len(word) <= 2:
        output += word
    else:
        output += (word[:1] + str(len(word) - 2) + word[-1:])
print(output)
```
