# Quiz 6

 Given a string, create a program that produces the sum of the characters in the string.

## code

```py
string = input("enter string: ")
sum = 0

for i in string:
    sum += ord(i.lower()) - 96

print(f'the sum of your ascii = {sum}')                                                                                
```
