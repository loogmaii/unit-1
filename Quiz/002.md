# Quiz 2

## given 2 numbers, A and B, output true if one of them is 20 or their sum is 20 (HL:array)


## code
```py
list_a = [10, 30, 10, 26]
list_b = [20, 15, 5, -6]
output = False

for i in range(len(list_a)):
    if list_a[i] == 20 or list_b[i] == 20:
        output = True

print(output)
```
